Servlet
 - doGet : 최초진입(+조회) -> jsp 호출
 - doPost : db에 데이터 저장(jsp에서는 수정) -> jsp호출 //jsp가 두개밖에 지원안함
 - doPut : 원래있던 데이터 수정
 - doDelete : 데이터 삭제
